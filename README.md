# 📊 SQL Data Analysis Project

## 🔹 Overview
This project analyses customer orders and spending behaviour using SQL.  
It focuses on extracting insights from transactional data, including customer performance, spending patterns, and regional trends.

---

## 🛠 Tools
- SQL

---

## 📈 Key Analysis
- Total spending by country  
- Customer order frequency and value  
- Top 5 customers by order volume  
- Average spending by city  
- High-value transactions  
- Highest spending per customer (subquery)  

---

## 🧠 Skills Demonstrated
- Joins (INNER JOIN)  
- Aggregations (SUM, COUNT, AVG)  
- Filtering (WHERE)  
- Grouping (GROUP BY)  
- Subqueries  
- Data analysis for business insights
